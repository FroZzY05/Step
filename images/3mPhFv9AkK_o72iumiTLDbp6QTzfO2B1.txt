В проекті MolotokMvc передбачені два типи користувачів - user та company.
Після логіну всі користувачі потрапляють на одну і ту ж сторінку Index. 
Ця сторінка повинна відображати як профіль user, так і профіль company.
Для user тут має відображатись список resume, для company - список vacancy.
Вам потрібно створити контролер ResumeController (по зразку VacancyController) з методами Create для додавання нових резюме і передбачити на сторінці Index
відображення списку resume для користувачів user.
